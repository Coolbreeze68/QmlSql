QmlSqlCreateDatabase QML Type

Creates a local database for sqlight ONLY. One must set the databaseName and also the filePath in which they would like to save the database file. One can use a md5sum that is randomly generated to create the databaseName. Or if you like you can also name the database fileName. The method of exec is what is used to create the database. More...

Detailed Description

Example:


  QmlSqlCreateDatabase{
          id: mainCreateDatabase
          fileName: "SomeFileName"
          databaseName: "ANewDatabase"
          filePath:  "SomePath"
          useMd5: false
          onCreated:{
              //do something
          }
   }

Property Documentation

databaseName : string

sets the name of the sqlight database


errorString : string

Returns a error string if there is a error in the creation of the database.


fileName : string

Sets the file name of the sqlight database

See also useMd5.


filePath : string

Sets the path to where one would like to create the sqlight database


lastCreatedDatabaseFile : string

returns a string of the last sqlight database that was created

See also useMd5.


useMd5 : bool

Sets this to true if you would like the fileName of the sqlight database to be a randomly gentorated md5 sum. One can not use both fileNmae and use this feature.

see also : fileName


Method Documentation

void exec()

A method that is run to create the database. Returns a error if it can not complete the method. If one is using a md5 it creates the filename based on the random md5 that is generated in this method

See also useMd5.
