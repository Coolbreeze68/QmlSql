QmlSql - Query a database and get results back in a string

This Example show how to open up a model and then query it mulitple times with Qml

The example needs you to fill in some using info about the database that you are querying

Note: passwords are set in plain text if you wouldn't like that please see the documentation on QmlSqlDatabase password

Running the Example

To run the example from Qt Creator Manual, open the Welcome mode and select the example from Examples. For more information, visit Qt Creator: Building and Running an Example}.
